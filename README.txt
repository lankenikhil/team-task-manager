# 📌 Assignment Overview

Build a web application where users can:

- Create projects
- Assign tasks
- Track project progress
- Manage teams with role-based access

### 👥 Roles
- **Admin**
- **Member**

# 🛠️ Tech Stack

## Frontend
- Next.js 15
- TypeScript
- Tailwind CSS
- Shadcn UI

## Backend
- Next.js API Routes
- MongoDB Atlas
- Mongoose

## Authentication
- JWT Authentication
- HTTP-only Cookies
- bcryptjs

## Deployment
- Vercel

---

# ⚙️ Environment Variables

Create a `.env.local` file:

```env
JWT_SECRET=your_secret_key

MONGODB_URI=your_mongodb_connection_string

NEXT_PUBLIC_APP_URL=http://localhost:3000
```

# 🚀 Getting Started

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/team-task-manager.git
```

---

## 2️⃣ Install Dependencies

```bash
npm install
```

---

## 3️⃣ Run Development Server

```bash
npm run dev
```

---

## 4️⃣ Open Browser

```bash
http://localhost:3000
```
